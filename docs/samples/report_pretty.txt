Sample output from --report-pretty flag
=========================================

This demonstrates the human-readable report format for a simple pipeline run.

Command:
--------
python -m mdp test_input.md --steps mask,prepass-basic,prepass-advanced --report report.json --report-pretty


Output:
-------

====================================================================================================
                                            RUN SUMMARY                                             
====================================================================================================
  Input file     : testing/test_data/test_input.md
  Output file    : output.md
  Pipeline steps : mask -> prepass-basic -> prepass-advanced

====================================================================================================
                                          PHASE STATISTICS                                          
====================================================================================================
  Mask             : 2 regions masked
  Prepass Basic    : 3 normalizations
  Prepass Advanced : 11 normalizations

====================================================================================================
                                             ARTIFACTS                                              
====================================================================================================
  Output file : output.md

Notes:
------
- Sections are only shown when relevant data exists
- File paths are truncated to ~60 chars for readability
- Growth ratios show file size changes with percentages
- Rejection counts help diagnose model output issues
- Determinism flags indicate if seed was set for reproducibility
