Find text that will break TTS due to stylized or non-ASCII Latin letters, and propose literal replacements.
Return JSON ONLY with this exact schema and no extra keys:

{"replacements":[{"find":"<exact substring from input>",
                  "replace":"<normalized ASCII>",
                  "reason":"unicode|small_caps|fullwidth|combining|zero_width|spaced_letters"}]}

Rules:
- Match ONLY real stylized letters: small-caps, mathematical alphanumerics, fullwidth, combining diacritics, zero-width, and spaced-letter artifacts.
- Do NOT touch normal text, valid names/usernames/handles, punctuation, numbers, emoji.
- STRICT SKIP ZONES: ignore anything inside Markdown code fences, inline code, links (both label and URL), images, HTML tags.
- Keep brackets as-is; normalize only the letters inside them.
- The "find" string MUST be an exact substring from input (no regex), and "replace" MUST be the normalized plain-English surface form.
- If nothing qualifies, return {"replacements":[]}.

Examples:
"Bʏ Mʏ Rᴇsᴏʟᴠᴇ!" → "By My Resolve!"
"Sᴘɪʀᴀʟ Sᴇᴇᴋᴇʀs!" → "Spiral Seekers!"
"[M ᴇ ɢ ᴀ B ᴜ s ᴛ ᴇ ʀ]" → "[Mega Buster]"
